# Outil de détection deepfake pour image/vidéo 

Front-end permettant :
- de sélectionner ou glisser/déposer une image
- de prévisualiser l'image
- de lancer l'outil pour savoir si l'image/vidéo est un deepfake, avec un indicateur chiffré


**Remarque:** le back-end n'étant pas encore disponible, seulement une simulation est possible pour le moment
